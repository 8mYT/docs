# Introduction  

ProBot is a highly configurable, multipurpose bot for greeting images, detailed logs, social commands, music, moderation, self-assignable roles, social media notifications, and protection.

## About our docs

This documentation will provide an in-depth explanation of how to use and configure each feature and how to troubleshoot any issues that may arise.

## Modules 

By clicking on a module's name in the sidebar on the left, you will be immediately redirected to that module's configuration page.

## Commands 

All probot commands have their page along with command usage and examples, which can be seen through the [link](https://probot.io/commands).
