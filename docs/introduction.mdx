# Introduction 

ProBot is a highly configurable, multipurpose bot that offers features like image greeting, detailed logs, social commands, music, moderation, self-assignable roles, social media notifications, and protection for your server.

## About Our Docs

Our documentation provides comprehensive explanations on how to use and configure each feature of ProBot, as well as troubleshooting tips for any issues that may arise.

## Modules

Easily navigate to a specific module by clicking its name in the sidebar on the left. Each module has its own configuration page.

## Commands 

All ProBot commands have a dedicated page with usage examples and information. Check out our [commands page](https://probot.io/commands) for more details.
