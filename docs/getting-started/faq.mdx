# Frequently Asked Questions

This FAQ section is designed to provide answers to common questions that users have about the bot. The FAQ is divided into three main categories: General FAQ, Moderation FAQ, and Premium FAQ.

## General FAQ

<details>

<summary>Can I change the bot's prefix?</summary>

- Unfortunately, the bot no longer uses prefixes. Instead, we have implemented the use of discord slash commands. However, users can set aliases for their favorite commands and use them as an alternative.

</details>

<details>

<summary>Can I play YouTube music links?</summary>

- Due to YouTube's policy, we no longer support YouTube links. Currently, we only support Soundcloud, Bandcamp, and Viemo links.

</details>

<details>

<summary>Can I change the bot's timezone?</summary>

- At this time, it is not possible to change the bot's timezone. The default timezone is set to GMT+3.

</details>

<details>

<summary>Can I give specific users access to dashboard?</summary>

- Only users with the Administrator permission can have access to the dashboard. To grant access to specific users, create a role and assign it to them.

</details>

<details>

<summary>Can I use Probot for free?</summary>

Yes, Probot can be used for free. However, some features may require a premium subscription to access. You can check the pricing and subscription options on the [Premium Page] (https://probot.io/premium).

</details>

<details>

<summary>Is there a way to report bugs or suggest new features for Probot?</summary>

Yes, you can report bugs or suggest new features by joining the official [Probot Support] (https://discord.gg/probot) server, where you can share your feedback with the developers and other members of the community.

</details>

## Moderation FAQ

<details>

<summary>What is the default duration for actions?</summary>

- If no time is specified, the default duration for actions is three months.

</details>

<details>

<summary>What permissions do my users need to use the moderation commands?</summary>

- The permissions required for each moderation command vary. To determine the necessary permission, users can try to use the command and receive an error message indicating the required permission.

</details>

<details>

<summary>Can I give users access to use moderation commands without giving them permission?</summary>

- Yes, it is possible to give users access to use moderation commands without giving them permission. The bot has an option called Mod Roles which allows users to specify roles that are allowed to use all moderation commands.

</details>

## Premium FAQ


<details>

<summary>How can I cancel my premium subscription?</summary>

- You can cancel your premium subscription by going to the manage premium page on the bot's website and selecting the `Cancel Subscription` option for the desired bot. Once the subscription is cancelled, you will no longer be charged and your premium features will be deactivated once the bot's duration expires. 

</details>

<details>

<summary>What happens if I downgrade from premium to free version?</summary>

- If you downgrade from the premium version to the free version, you will lose access to all premium features. Your data will not be deleted, but it will be locked until you upgrade again.

</details>

<details>

<summary>Is there a free trial for the premium version?</summary>

- Unfortunately, there is no free trial for the premium version. 

</details>

<details>

<summary>Do I get a refund if I am not satisfied with the premium version?</summary>

- Refunds will be considered on a case-by-case basis. If you would like to request a refund, please contact our support team with the reason for your request.

</details>
