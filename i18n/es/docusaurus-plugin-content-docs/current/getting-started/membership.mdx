# Membresías

La función de membresía proporciona a los usuarios ventajas y privilegios exclusivos que no están disponibles para los usuarios sin ella. Normalmente, los usuarios deben pagar una cuota para acceder a estas ventajas, que puede ser una suscripción mensual o anual.

Hay tres niveles de afiliación disponibles:

- **Plata**
- **Oro**
- **Diamante**

Cada nivel ofrece distintas ventajas y está diseñado para satisfacer las necesidades de usuarios con distintos presupuestos y requisitos de uso.

## Nivel de Membresía Plata

El nivel de membresía Plata es el nivel básico y ofrece a los usuarios acceso a las siguientes ventajas:

- (x3) recompensas diarias, esto puede ser hasta 10.000 créditos en diario
- 5 reputaciones por día
- Fondo personalizable (Perfil / ID)

:::note
Este nivel de membresía es ideal para usuarios con necesidades relativamente modestas y que buscan una forma económica de acceder a algunas de las ventajas de la membresía.
:::

## Nivel de Membresía Oro

El nivel de membresía Oro es el nivel medio y ofrece a los usuarios acceso a las siguientes ventajas:

- (x5) recompensas diarias, esto puede suponer hasta 25.000 créditos diarios
- 10 reputaciones por día
- Fondo personalizable (Perfil / ID)
- Una transferencia de cuenta al mes

:::note
Este nivel de membresía está diseñado para usuarios que tienen necesidades más importantes y requieren un nivel de funcionalidad superior al que puede ofrecer el nivel Plata.
:::

## Nivel de Membresía Diamante

El nivel de membresía Diamante es el nivel superior y ofrece a los usuarios acceso a las siguientes ventajas:

- (x10) recompensas diarias, esto puede suponer hasta 50.000 créditos diarios
- 30 reputaciones por día
- Fondo personalizable (Perfil / ID)
- Una transferencia de cuenta al mes

:::note
Este nivel de membresía es ideal para usuarios avanzados que requieren un alto nivel de funcionalidad y acceso a ventajas exclusivas.
:::

## Daily on Discord

The Daily feature on our Discord platform allows users to receive a daily credit amount. The amount of credits awarded varies depending on the user's membership tier, as explained in each membership tier description.

## Daily Reps

The Daily Reps feature enables users to give reputation points to their friends through the use of the `/rep` command. These points will be displayed on the friend's profile.

## Change Background

The Change Background feature provides users with the option to upload their own unique background image for their profile and ID.

:::note
It's important to note that supported file types are limited to `png`, `jpeg`, and `SVG`, and that images should not exceed `3MB` in size.
:::

:::warning
Inappropriate backgrounds or content, including `+18` and `NSFW` content, are strictly prohibited. Violation of these guidelines may result in the removal of a user's subscription and their ability to subscribe again.
:::

## Transfer Per Month

The Transfer Per Month feature allows users to transfer their old account data, including their level and credits, once per month. To complete this process, users will be asked to provide their old account ID and email address. It's crucial to note that the email address provided must match the email address associated with the old account. Users will receive a 6-digit authentication code via email to complete the transfer process.

:::note
It's worth noting that the ability to transfer profile data is only available once per month. This restriction ensures that the transfer process is fair and prevents abuse of the feature.
:::

:::note
It's important to keep in mind that the authentication code is only valid for 15 minutes. If users don't receive a code within that timeframe, they will need to check their email and spam folder or wait for another code.
:::
