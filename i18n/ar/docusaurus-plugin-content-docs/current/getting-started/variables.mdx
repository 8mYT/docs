# متغيرات لوحة تحكم بروبوت

لإنشاء رسالة منسقة و إحترافية و مثالية يمكنك إستخدام متغيرات مختلفة لفعل ذلك، سيوفر لك هذا الدليل الشامل جميع المتغيرات التي يمكنك إستخدامها في الأقسام المختلفة في لوحة التحكم.

## المتطلبات

قبل إستخدام المتغيرات يجب عليك تفعيل وضع الديسكورد ديفلوبر في جهازك وهذا سوف يسمح لك بالحصول على الأيدي الذي تحتاجه لفعل اشياء محددة.

## تفعيل وضع الديسكورد ديفلوبر  

> على الكمبيوتر
1. قم بفتح برنامج الديسكورد
2. إنتقل الى `User Settings`
3. ثم إلى `App Settings` وقم ر `Appearance`
4. فعل وضع الديفلوبر 

> على الهاتف
1. فتح الديسكورد
2. انتقل الى `User Settings`
3. مرر الى `App Settings` — انقر `Behavior`
4. فعل وضع الديفلوبر 


بمجرب تفعيل وضع ديسكورد ديفلوبر فأنت جاهز لاستخدام المتغيرات.

## متغيرات الترحيب

يمكن استخدام المتغيرات الاتية في قسم الترحيب بـ لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[user]` | لمنشن العضو الجديد. | 
| `[inviter]` | لمنشن الداعي. |
| `[server]` | لذكر اسم السيرفر. |
| `[userName]` | لذكر اسم العضو الجديد بدون منشن. |
| `[inviterName]` | لذكر اسم الداعي بدون منشن. |
| `[memberCount]` |  لذكر عدد اعضاء السيرفر بدون منشن. |
| `<@user_id>` | لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` | لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` | لمنشن رول محدد باستخدام الايدي. |

## متغيرات الرد التلقائي

يمكن استخدام المتغيرات الاتية في قسم الرد التلقائي بـ لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[user]` | لمنشن كاتب رسالة الرد. |
| `[userName]` |  لذكر اسم كاتب رسالة الرد بدون منشن. |
| `[invites]` |  لعرض عدد دعواتك. |
| `<@user_id>` |  لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` | لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` | لمنشن رول محدد باستخدام الايدي. |

## متغيرات رسائل الايمبد

يمكن استخدام المتغيرات الاتية في قسم رسائل الايمبد بـ لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[text](link)` | هذا المتغير لربط نص بـ رابط محدد عند الضغط على النص ستنتقل الى الرابط. |
| `<@user_id>` |  لمنشن عضو محدد باستخدام الايدي. |
| `<#channel_id>` |  لمنشن روم محدد باستخدام الايدي. |
| `<@&role_id>` |  لمنشن رول محدد باستخدام الايدي. |

## متغيرات نظام اللفلات 

يمكن استخدام المتغيرات الاتية في قسم نظام اللفلات بـ لوحة التحكم:

| المتغير | الوصف | 
| ------------- |:-------------:|
| `[level]` |  لذكر اللفل الحالي للعضو. | 
| `[user] `| لمنشن العضو. |
| `[user.username]` | لذكر اسم العضو بدون منشن. |
| `[user.tag]` | لذكر تاق العضو. |
| `[oldLevel] ` | لذكر اللفل السابق للعضو. |

باستخدام المتغيرات المذكورة اعلاه ستجعل رسالتك تبدو مثالية و احترافية.
