# Команды

В этом разделе вы лучше поймёте команды ProBot, их применение и примеры использования.

:::важно

- Так как слеш является префиксом бота по умолчанию, все примеры использования будут на примере слеш-команд `/`.

- Знаки `[` `]` обязательны для каждой команды, на которую ссылается этот знак, а `(` `)` необязательны.

:::

## Общее 

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /credits | Показать ваши или чьи-либо кредиты | /credits [пользователь] (число) |
| /rep | Наградить кого-нибудь очком репутации. Можно использовать раз в 24 часа | /rep [пользователь] |
| /moveme | Перемещает вас в другой голосовой канал | /moveme [канал или пользователь] |
| /color | Смените ваш цвет на сервере | /color [номер цвета] |
| /colors | Показывает список доступных цветов | /colors |
| /short | Сокращает URL | /short [URL] |
| /roll | бросок игральных костей | /roll (число) |

## Система уровней

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /profile | Посмотреть ваш или чей-либо кастомизируемый персональный общедоступный профиль | /profile (пользователь) |
| /rank | Посмотреть вашу или чью-либо рейтинговую карточку на сервере | /rank (пользователь) |
| /top | Отобразить топ участников по тексту или войсу | /top (day/week/month) |
| /title | Меняет заголовок вашего профиля | /title [новый заголовок] |
| /setxp | Устанавливает опыт пользователя | /setxp [пользователь] [тип] [новый_опыт] |
| /setlevel | Устанавливает уровень пользователя | /setlevel [пользователь] [тип] [новый_уровень] |

## Информация

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /user | Показывает информацию о себе или пользователе, например ID и дату вступления | /user (пользователь) |
| /avatar | Получить аватар пользователя | /avatar (пользователь) |
| /server | Показывает информацию о сервере | /server |
| /roles | Получить список ролей сервера и число участников | /roles |

## Модерация

| Команда | Описание | Использование |
| ------------- |:-------------:|:-------------:|
| /setnick | Меняет никнейм участника | /setnick [пользователь] (никнейм) |
| /ban | Банит участника | /ban [пользователь] (время) (причина) |
| /unban | Снимает бан с участника | /unban [пользователь или его ID] |
| /kick | Кикает участника | /kick [пользователь] (причина) |
| /vkick | Кикает участника с голосового канала | /vkick [пользователь] |
| /mute text | Замутить участника, вследствие чего он не сможет печатать в текстовых каналах | /mute text [пользователь] (время) (причина) |
| /unmute text | Снимает мут с участника | /unmute [пользователь] |
| /mute voice | Замутить участника, вследствие чего он не сможет говорить в голосовых каналах | /mute voice [пользователь] (время) (причина) |
| /unmute voice | Снимает голосовой мут с участника | /unmute voice [пользователь] |
| /timeout | Отправить участника в тайм-аут | /timeout [пользователь] (время) (причина) |
| /untimeout | Убирает тайм-аут с участника | /untiemout [пользователь] |
| /clear | Очищает сообщения канала | /clear (пользователь/боты) (количество сообщений) | 
| /move | Перемещает участника в другой голосовой канал | /move [пользователь] (пользователь или канал) |
| /role give | Добавить/снять роль(и) с участника | /role give [пользователь] [роль] (bulk) |
| /role remove | Снять роль(и) с участника | /role remove [пользователь] [роль] (bulk) |
| /role multiple | Добавить/снять роль(и) с участников | /role multiple [добавить_или_снять] [роль] [тип_выбора] (необходимая роль) |
| /points set | Установить баллы пользователя | /points set [пользователь] [баллы] |
| /points list | Посмотреть все баллы | /points list (роль) (страница) |
| /points decrease | Уменьшить баллы пользователя | /points decrease [пользователь] [баллы] |
| /points increase | Добавить баллы пользователя | /points increase [пользователь] [баллы] |
| /points reset | Reset the points | /points reset (user) |
| /warn | Warns a member | /warn [user] [reason] |
| /warn_remove | Remove warnings for the server or user | /warn_remove [user/all/warnID] |
| /warnings | Get the list of warnings for the server or a user | /warnings (user) |
| /lock | Disables `@everyone` from sending messages in specific channel | /lock (channel) (reason) |
| /unlock | Allows `@everyone` to send messages in specific channel | /unlock (channel) |
| /setcolor | Changes role's color by hex codes | /setcolor [role name] [color code]
| /slowmode | Enable or disable slowmode on a channel | /slowmode (time) |
| /reset | Reset text/voice/invites xp points for all or specific members | /reset [text / voice / invites / limits) [all / user] |

<details>

<summary>Important Moderation Notes</summary>

- Only users with `Administrator` permission, `Mod Roles` on the dashboard, or the permission for that command can use moderation commands. __For example__, users with ban permission can use the `/ban` command.

- If no time was specified, the default time duration for all time-input commands is **3 months**; this can be modified at any time. 

</details>
