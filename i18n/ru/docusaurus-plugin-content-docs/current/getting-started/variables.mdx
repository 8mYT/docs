# Переменные панели управления ProBot

Чтобы ваши сообщения выглядели более профессионально и эстетично, вы можете использовать различные переменные для достижения этой цели. Это руководство покажет все переменные, которые можно использовать в различных разделах панели управления.

## Необходимые условия

Перед использованием переменных, вы должны включить Режим разработчика в Discord на вашем устройстве. В этом режиме вы получите ID, необходимые для упоминания конкретных вещей.

## Включение Режима разработчика Discord

> На ПК
1. Запустите Discord
2. Кликните на `Настройки пользователя`
3. Пролистайте до `Настройки приложения` — выберите `Расширенные`
4. Включите Режим разработчика

> На телефоне
1. Запустите Discord
2. Нажмите на `Настройки пользователя`
3. Пролистайте до `Настройки приложения` — выберите `Расширенные`
4. Включите Режим разработчика

После включения Режима разработчика вы готовы к использованию переменных.

## Переменные Добро пожаловать

Следующие переменные могут быть использованы в разделе Добро пожаловать в вашей панели управления Discord:

| Переменная | Описание | 
| ------------- |:-------------:|
| `[user]` | Эта переменная упоминает нового участника | 
| `[inviter]` | Эта переменная упоминает приглашающего |
| `[server]` | Эта переменная отображает имя сервера |
| `[userName]` | Эта переменная отображает имя нового участника |
| `[inviterName]` | Эта переменная отображает имя приглашающего |
| `[memberCount]` | Эта переменная отображает достигнутое количество участников |
| `<@user_id>` | Эта переменная упоминает определённого пользователя, используя его ID |
| `<#channel_id>` | Эта переменная упоминает определённый канал, используя его ID |
| `<@&role_id>` | Эта переменная упоминает определённую роль, используя её ID |

## Auto Responder Variables

The following variables can be used in the Auto Responder section of your Discord dashboard:

| Variable | Description | 
| ------------- |:-------------:|
| `[user]` | This variable mentions the member who typed the trigger |
| `[userName]` | This variable displays the name of the user who typed the trigger |
| `[invites]` | This variable displays the number of invites you have reached |
| `<@user_id>` | This variable mentions a specific user using their ID |
| `<#channel_id>` | This variable mentions a specific channel using its ID |
| `<@&role_id>` | This variable mentions a specific role using its ID |

## Embed Messages Variables

The following variables can be used in the Embed Messages section of your Discord dashboard:

| Variable | Description | 
| ------------- |:-------------:|
| `[text](link)` | This variable creates a hyperlink |
| `<@user_id>` | This variable mentions a specific user using their ID |
| `<#channel_id>` | This variable mentions a specific channel using its ID |
| `<@&role_id>` | This variable mentions a specific role using its ID |

## Leveling System 

The following variables can be used in the Leveling System section of your Discord dashboard:

| Variable | Description | 
| ------------- |:-------------:|
| `[level]` | to print given level | 
| `[user] `| to ping the user |
| `[user.username]` | to print user name |
| `[user.tag]` | to bring user tag |
| `[oldLevel] ` | to bring old user level |

By using the variables listed above, you can make your messages more professional and eye-catching.
