# Переменные панели управления ProBot

Чтобы ваши сообщения выглядели более профессионально и эстетично, вы можете использовать различные переменные для достижения этой цели. Это руководство покажет все переменные, которые можно использовать в различных разделах панели управления.

## Необходимые условия

Перед использованием переменных вы должны включить Режим разработчика Discord на вашем устройстве. В этом режиме вы сможете получить ID, необходимый для упоминания конкретных вещей.

## Включение Режима разработчика Discord

> На ПК
1. Запустите Discord
2. Кликните на `Настройки пользователя`
3. Пролистайте до `Настройки приложения` — выберите `Расширенные`
4. Включите Режим разработчика

> На телефоне
1. Запустите Discord
2. Нажмите на `Настройки пользователя`
3. Пролистайте до `Настройки приложения` — выберите `Расширенные`
4. Включите Режим разработчика

После включения Режима разработчика вы готовы к использованию переменных.

## Переменные Добро пожаловать

Следующие переменные могут быть использованы в разделе Добро пожаловать в вашей панели управления Discord:

| Переменная | Описание | 
| ------------- |:-------------:|
| `[user]` | Эта переменная упоминает нового участника | 
| `[inviter]` | Эта переменная упоминает приглашающего |
| `[server]` | Эта переменная отображает имя сервера |
| `[userName]` | Эта переменная отображает имя нового участника |
| `[inviterName]` | Эта переменная отображает имя приглашающего |
| `[memberCount]` | Эта переменная отображает достигнутое количество участников |
| `<@user_id>` | Эта переменная упоминает определённого пользователя, используя его ID |
| `<#channel_id>` | Эта переменная упоминает определённый канал, используя его ID |
| `<@&role_id>` | Эта переменная упоминает определённую роль, используя её ID |

## Переменные Автоответчика

Следующие переменные можно использовать в разделе Автоответчик в вашей панели управления Discord:

| Переменная | Описание | 
| ------------- |:-------------:|
| `[user]` | Эта переменная упоминает участника, который ввёл триггер |
| `[userName]` | Эта переменная отображает имя пользователя, который ввёл триггер |
| `[invites]` | Эта переменная отображает количество приглашений, которое вы достигли |
| `<@user_id>` | Эта переменная упоминает определённого пользователя, используя его ID |
| `<#channel_id>` | Эта переменная упоминает определённый канал, используя его ID |
| `<@&role_id>` | Эта переменная упоминает определённую роль, используя её ID |

## Переменные Embed-сообщений

Следующие переменные могут быть использованы в разделе Embed-сообщения в вашей панели управления Discord:

| Переменная | Описание | 
| ------------- |:-------------:|
| `[текст](ссылка)` | Эта переменная создаёт гиперссылку |
| `<@user_id>` | Эта переменная упоминает определённого пользователя, используя его ID |
| `<#channel_id>` | Эта переменная упоминает определённый канал, используя его ID |
| `<@&role_id>` | Эта переменная упоминает определённую роль, используя её ID |

## Система уровней

Следующие переменные могут быть использованы в разделе Система уровней в вашей панели управления Discord:

| Переменная | Описание | 
| ------------- |:-------------:|
| `[level]` | для отображения выданного уровня | 
| `[user] `| для упоминания пользователя |
| `[user.username]` | для отображения имени пользователя |
| `[user.tag]` | для отображения тэга пользователя |
| `[oldLevel] ` | для отображения старого уровня пользователя |

Используя перечисленные выше переменные, вы можете сделать ваши сообщения более профессиональными и привлекательными.
