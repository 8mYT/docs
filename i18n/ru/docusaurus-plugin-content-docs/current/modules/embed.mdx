# Embed-сообщения

Функция ярких embed-сообщений позволяет вашему боту отправлять кастомизированные версии ваших предпочитаемых сообщений. В этом руководстве вы узнаете, как установить и настроить эти embed-сообщения.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Создание нового Embed-сообщения 

Чтобы создать новое сообщение, нажмите на кнопку `Создать`. Это перенаправит вас на новую страницу, где вы можете увидеть предосмотр Embed-сообщения и внести необходимые корректировки. Также эта страница содержит несколько кнопок:

- Отправить в канал 
- Редактировать отправленное сообщение
- Сохранить изменения
- Отменить изменения

## Настройка Embed-сообщения

На новой странице вы увидите дополнительную кнопку, помеченной как `Добавить поле`, которое содержит все варианты кастомизации вашего embed-сообщения. К этим вариантам относятся:

- Заголовок
- Описание
- URL
- Изображение
- Миниатюра
- Автор
- Цвет
- Футер
- Поле

После выбора желаемых вариантов вам будет предоставлено поле для текста, где вы можете ввести ваш текст сообщения.

## Отправка Embed-сообщения

Как только вы настроили ваше Embed-сообщение, вы можете использовать кнопку `Отправить на канал` для отправки сообщения в желаемый канал.

## Сохранение изменений

Чтобы сохранить внесённые изменения в ваше embed-сообщение, воспользуйтесь кнопкой `Сохранить изменения`.

## Editing a Sent Message

If you need to edit an existing message that has already been sent, you can use the `Edit Message` button.

## Canceling Changes 

If you need to undo any changes made to your embed message, use the `Cancel Changes` button

:::caution

Please note that there are limits to the Discord embed feature, so if the bot is unable to send the embed message, you may have exceeded the limit. To avoid this, be sure to completely fill out every required field and pay attention to the following limits:

- Embed title is limited to `256` characters
- Embed description is limited to `4096` characters
- An embed can contain a maximum of `25` fields
- A field name/title is limited to `256` character and the value of the field is limited to `1024` characters
- Embed footer is limited to `2048` characters
- Embed author name is limited to `256` characters
- The total of characters allowed in an embed is `6000`

:::

By adhering to these limits, you can ensure that your bot is able to send the embed message without any issues
