# Автороли

Функция Автороль ProBot позволяет автоматически назначать роли новым пользователям на их сервере Discord. В этом руководстве вы пройдете через шаги по установке и настройке этой функции.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Настройка

Получив доступ к модулю Автороли, вы можете начать настройку ролей, которые будут автоматически назначаться новым пользователям. Для этого воспользуйтесь меню выбора модуля, чтобы выбрать одну или несколько ролей по желанию.

## Premium Auto-Role

You can set a unique link for auto and particular roles if you are a premium subscriber.

1. Select `Add`
2. Then, click the `Edit` button.
3. Insert the invitation link
4. Select the roles you'd like members to have if they join through this invite link.

:::caution

1. The bot's role must be higher than the auto-role chosen.

2. If the Discord rule-screen panel is enabled, newly-joined users will be required to accept the rules before receiving the auto role.

3. The bot will not assign users roles with administrative privileges.

:::

By following these steps and taking note of the important notes, you can successfully set up and configure the Auto-Roles feature on your Discord server using ProBot.
