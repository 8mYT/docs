# Система уровней

С системой уровней ProBot активные пользователи будут автоматически награждены необычными уровневыми ролями, имеющими эксклюзивные права и доступ к тайным каналам, как только они достигнут указанного уровня!

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

Сначала у вас будет три различных варианта, каждый из которых можно настроить для получения более индивидуального уровнего опыта.

## Роли, не получающие XP

Используя это, вы можете запретить определенным пользователям с выбранными ролями накапливать очки опыта в любом месте вашего сервера.

## Каналы без получения XP

Вы можете запретить определенным каналам накапливать очки опыта.

## Канал повышения уровня

Отправляйте поздравительные сообщения пользователям, которые поднялись с одного уровня на более высокий. Вы также можете использовать уникальные переменные в этих сообщениях!

:::caution

**Сообщение о повышении уровня** будет отправлено только тем, кто повысил текстовый уровень. В настоящее время у нас нет сообщения о голосовом уровне.

:::

## Настройка

Процедура создания системы вознаграждения ролью описана ниже:


Появится кнопка `Добавить награду ролью`, где вы можете добавить награду ролью, свой уровень, которого должны достичь пользователи, и другие параметры, описанные ниже.

### Текстовый уровень

- Это текстовый уровень, которого должен достичь участник, чтобы получить роль за уровень, если она была установлена самостоятельно.

- Диапазон тестовых уровней — от 0 до 100; предоставляйте пользователям дополнительные вознаграждения по мере достижения ими необходимого уровня.

### Голосовой уровень

- Это головой уровень, которого должен достичь участник, чтобы получить роль за уровень, если она была установлена самостоятельно.

- Диапазон голосовых уровней — от 0 до 100; предоставляйте пользователям дополнительные вознаграждения по мере достижения ими необходимого уровня.

### Role Rewards

Reward the members with exceptional level positions as they progress through the levels and meet your requirements.

:::caution

If the roles do not display, the bot's role is nested underneath all server roles. To resolve this issue, move the bot role to the top of the roles hierarchy to ensure that roles are displayed.

:::

## Remove with a higher level

It is entirely up to you to decide whether the levels should stack or be removed when users climb to higher levels.

- If you disable the option, roles will stack.

- If you enable the option, roles get removed as a higher level is reached.

## DM Member

Notifies a member when they receive a text-level role via direct message.

## Important Notes

:::important Three potential situations that may arise when setting up a level requirement

1. **Both Voice & Text**: If you set the need for both the text and voice level without leaving `0` in any option, both Text and Voice will be required. This will require the user to reach both levels to obtain the role reward.

2. **Text only**: If you set the need for the text level and leave the Voice at `0`, only Text will be required. This will require the user to reach the level needed for the Text to obtain the role reward.

3. **Voice only**: If you set the need for the voice level and leave the Text at `0`, only Voice will be required. This will require the user to reach the level needed for the Voice to obtain the role reward.

4. **Role Position**: If you have selected the same role rewards for many levels, don't worry—it will still provide users with the role you specified as they reach the requirements of any level you set for the same role.

:::

:::caution

Users need to type in chats for the bot to update their XP data and award them voice-level prizes, and only then will the bot be able to reward them.

:::
