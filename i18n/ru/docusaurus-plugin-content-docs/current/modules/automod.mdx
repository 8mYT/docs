# Автомодерация

Обеспечьте бесперебойную работу сервера без стресса с помощью ProBot Automod. Попрощайтесь со спамерами и рекламщиками, и наслаждайтесь спокойствием, зная, что ProBot неустанно работает над поддержанием безопасной и надежной среды для вашего сообщества.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите в панель управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

## Опции обнаружения

Для обнаружения потенциального спама или вредоносных сообщений на вашем сервере доступны следующие опции:

> **СПАМ (5 СООБЩЕНИЙ/5 СЕК)**: Эта опция определяет, когда пользователь отправляет более 5 сообщений за 5-секундный промежуток.

> **ПЛОХИЕ СЛОВА**: Эта опция определяет, содержит ли сообщение плохие слова.

> **ДУБЛИРОВАННЫЙ ТЕКСТ**: Эта опция определяет сообщения, содержащие дублированный текст.

> **ПОВТОРЯЮЩИЕСЯ СООБЩЕНИЯ**: Эта опция определяет сообщения, которые повторяют текст или строки, например "СПАМ СПАМ СПАМ СПАМ".

> **DISCORD-ПРИГЛАШЕНИЯ**: Эта опция определяет сообщения, содержащие инвайт-ссылки Discord.

> **ССЫЛКИ**: This option detects messages that contain links.

> **СПАМ КАПСОМ (70% > КАПС)**: This option detects messages that are written in all caps, like "HELLO THIS IS PROBOT."

> **СПАМ EMOJI**: This option detects messages that contain eight or more emojis.

> **МАССОВОЕ УПОМИНАНИЕ**: This option detects messages that mention a large number of users.

## Action Options

Each detection option offers three different action options to meet the specific needs of your community. These actions are:

1. **Delete Message**: Deletes the message that triggered the detection.

2. **Mute Member**: Mutes the member who sent the triggering message.

3. **Warn Member**: Issues a warning to the member who sent the triggering message.

## Settings

Each detection option features a **Settings** option, indicated by a gear icon `⚙️`. Clicking on this option will reveal a range of settings that you can adjust to optimize the use of ProBot Automod for your server.

> **Disabled Channels**: Channels that will be exempt from the detection options.

> **Disabled Roles**: Roles that will be exempt from the detection options.

> **Mute Duration**: The length of time that a member will be muted if they violate the detection options.

> **Mute Violations Count**: The number of violations required before a member is muted.

In the **Bad Words** and **Links** automation options, there are additional settings not found in the default options:

> **Bad Words**

- Bad Words: A list of individual bad words that will trigger the detection option.

- DON'T MATCH IF IT IS PART OF A WORD: If enabled, this option will prevent the detection of words that contain the bad words but are not identical to them.

> **Links**

- Whitelisted URL: A list of URLs that will not trigger the detection option, even if they contain links.

- Blacklisted URL: A list of URLs that will trigger the detection option, even if they do not contain links.

## Additional Features

In addition to the detection and action options described above, ProBot Automod also includes the following features:

> **Ignored Channels**: Channels that will be exempt from all detection options.

> **Ignored Roles**: Roles that will be exempt from all detection options.

> **Only Images Channel**: Allows you to enforce that only images are sent in specific channels.

> **Only YT Channel**: Allows you to enforce that only YouTube links are sent in specific channels.
