# Временные ссылки

Функция команды Временной ссылки позволяет администраторам сервера установить команду для пользователей для приглашения друзей с лёгкостью. Следующие инструкции содержат пошаговое руководство по настройке команды.

:::info

Панель управления — это место, где осуществляется управление описанными здесь конфигурациями.

1. Перейдите к панели управления и авторизуйтесь. 
2. Выберите ваш сервер в левом меню.
3. Выберите модуль и убедитесь, что он включён. 

:::

:::caution

Эта функция доступна только для премиум-подписчиков.

:::

## Название команды

Первым шагом является ввод имени команды, которую пользователи могут ввести, чтобы получить персональную ссылку. Команда может быть желаемым текстом, префиксом, символом и буквой.

## Сообщения

### Сообщение, когда ссылка была запрошена

Когда пользователь вводит команду, сообщение появится как ответ, указывая на то, что ссылка была успешно отправлена в ЛС пользователю. Текст этого сообщения может быть изменён по желанию.

### Сообщение, когда ссылка не может быть отправлена

Если бот не может отправить ссылку, отобразится отдельное сообщение в ответ, указывая на то, что бот не смог отправить ссылку, потому что ЛС пользователя закрыт. Текст этого сообщения также может быть изменён по желанию.

## Роли

### Запрещённые роли 

Администратор может предотвратить указанным ролям использование команды для ссылки.

### Разрешённые роли
 
Пользователи с указанными ролями смогут использовать команду для ссылки.

## Каналы

### Отключённые каналы

Бот не будет отвечать на команду ссылки в указанном канале.

### Разрешённые каналы

Бот ответит на команду ссылки в указанном канале.

## Информация о ссылке

### Сообщение текста со ссылкой

Сообщение, отправляемое в ЛС пользователю, когда он использовал команду, включая инвайт-ссылку с информацией об использовании ссылки и временем истечения. Текст этого сообщения может быть изменено по желанию.

### Использования ссылки

Администратор может указать, сколько раз можно использовать ссылку и лимит пользователей, которые могут присоединиться с этой ссылкой. Для установки количества использований ссылки предусмотрена полоса прокрутки.

### Канал приглашения по умолчанию

Администратор может указать, какой канал будет принадлежать к инвайт-ссылке. Например, если администратор установил канала "Тест", пользователи, воспользовавшиеся ссылкой, будут перенаправлены на канал "Тест" после присоединения.

### Срок действия ссылки в минутах

Администратор может указать срок действия ссылки в минутах. Ссылка истечёт, если она использована за это время. Для установки срока действия ссылки предусмотрена полоса прокрутки.

## Время ожидания

### Время ожидания в часах

Администратор может установить время ожидания для пользователей на новый запрос ссылки.

### Сколько ссылок можно запросить до времени ожидания

Администратор может установить количество ссылок, которые можно запросить до времени ожидания.

### Waiting Time Message

When a user attempts to request a new link before the wait time has ended, a message will appear as a reply indicating that they should wait the specified amount of time before requesting a new link. The text of this message can be adjusted as desired.

## Disable Wait Time

The administrator can allow users with `Invite Members` permission to bypass the wait time and create links anytime.

## Delete User Command Invocation and Bot's Reply

The administrator can choose to have the bot delete the user's command and the bot's reply after five seconds or keep it persistent. Enabling the option will delete the message, and disabling it will keep them.

:::important

These are the following limits for our temp link options;

- Link age in minutes `1440`
- Wait time in hours `9999999999999`
- How many links can be requested before wait time `9999999999999`

:::
