# Введение

ProBot — это очень настраиваемый многоцелевой бот, который предлагает такие функции, как приветственные изображения, детализированные логи, социальные команды, модерация, роли самостоятельного назначения, уведомления с социальных сетей и защита для вашего сервера.

## О нашей документации

Наша документация содержит полные объяснения по использованию и настройке каждой функции ProBot, а также советы по устранению неполадок, которые могут возникнуть.

## Модули

С лёгкостью переключайтесь к нужному модулю, кликнув на его имя в левом боковом меню. Каждый модуль имеет собственную страницу конфигурации.

## Команды

Все команды ProBot имеют специальную страницу с примерами использования и информацией. Для получения более подробной информации посетите нашу [страницу команд](https://probot.io/commands).

:::tip

:::

:::
